\noindent
Tuần thứ hai của Module 4 tập trung vào các \textbf{thuật toán boosting} tiên tiến và \textbf{triển khai API}, cùng với việc áp dụng vào \textbf{dự án thực tế}. Module này giúp người học nắm vững gradient boosting từ cơ bản đến nâng cao và cách triển khai mô hình ML thành API production.

\vspace{1em}
\noindent
Cụ thể, blog tuần này sẽ bao gồm các chủ đề:

\vspace{1em}
\begin{enumerate}
    \item \textbf{Gradient Boosting – Học qua ví dụ tính tay}  
    Hiểu gradient boosting từ cơ bản thông qua các ví dụ tính toán chi tiết và minh họa trực quan.

    \item \textbf{Gradient Boosting – Nâng cao}  
    Các kỹ thuật tối ưu hóa, regularization và so sánh với các thuật toán ensemble khác.

    \item \textbf{XGBoost – Extreme Gradient Boosting}  
    Thuật toán XGBoost hiện đại, ưu điểm vượt trội và ứng dụng trong các cuộc thi data science.

    \item \textbf{FastAPI – Xây dựng API cho ML Models}  
    Triển khai mô hình ML thành API production-ready với FastAPI framework.

    \item \textbf{Data Project – PG\&E Energy Analytics Challenge}  
    Dự án thực tế dự báo tải điện sử dụng time-series analysis và machine learning.
\end{enumerate}

\vspace{1em}
\noindent\textbf{ Giá trị nhận được sau khi đọc Blog}
\begin{itemize}
    \item Hiểu và triển khai được Gradient Boosting từ cơ bản đến nâng cao.  
    \item Nắm vững thuật toán XGBoost và ứng dụng trong các dự án thực tế.  
    \item Biết cách xây dựng API cho mô hình ML với FastAPI.  
    \item Áp dụng được time-series analysis và feature engineering.  
    \item Triển khai dự án ML hoàn chỉnh từ EDA đến production.  
\end{itemize}
